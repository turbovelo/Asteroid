# Asteroid

This is a small program for the course of spacecraft technology.
In the scope of performing a pre-phase A study for an Asteroid sample return mission, this python tool aims to provide an estimate of deltaV necessary for various envisionned trajectory possibilites using the poliastro toolbox.

# Visualisation

Open out/orbit.html to see the plot of the trajectory. Data obtained from the Horizons API can be found in the src/data.json file.

# How to

To run, execute the command

```
$ python src
```
